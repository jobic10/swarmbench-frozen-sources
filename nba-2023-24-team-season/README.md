# NBA 2023-24 team-season snapshot

A byte-stable snapshot of all thirty NBA teams from the 2023-24 regular season, taken from the ESPN standings API (season=2024), frozen for stable agent fetches. Per-team: wins, losses, win percent, points for and against (season totals and per-game), point differential, playoff seed, and home, road, last-ten and streak records. Data from ESPN (public).
