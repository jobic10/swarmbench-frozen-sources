# MLB 2024 team-season snapshot

Byte-stable snapshot of all 30 MLB teams 2024 regular-season figures from the public MLB Stats API (statsapi.mlb.com), frozen for stable agent fetches. index.json lists teams + the per-team URL pattern; teams/<id>.json holds each team record, run differential, Pythagorean expected wins, and hitting/pitching stats; reference/pythagorean_method.json gives the formula. Data (c) MLB Advanced Media, from statsapi.mlb.com.
