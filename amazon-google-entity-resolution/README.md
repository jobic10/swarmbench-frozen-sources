# Amazon-GoogleProducts entity-resolution catalogs (frozen)

Two software-product catalogs for a cross-catalog reconciliation task.
Source: Leipzig DB Group benchmark "Amazon-GoogleProducts" (public research dataset).
- amazon_catalog.csv: 1363 products (id,title,manufacturer,price,description)
- google_catalog.csv: 3226 products (id,name,manufacturer,price)
The gold matching is intentionally NOT included here.
